# 📊 Power BI Satış Analizi Dashboardu – FINAL.pbix

Bu layihə Power BI vasitəsilə yaradılmış satış analizi dashboardudur. Burada müxtəlif satış göstəricilərinin interaktiv vizuallaşdırılması və təhlili həyata keçirilir.

## ✅ Əsas Dashboard Xüsusiyyətləri

🔸 **Ümumi Gəlir və Satış Sayı** – əsas performans indikatorları (KPI-lar)  
🔸 **Top 10 Məhsul** – ən çox satılan məhsulların sıralanması  
🔸 **Top Müştərilər** – ən çox alış edən müştərilərin siyahısı  
🔸 **Satış Regionları üzrə Parçalanma** – xəritə və ya sütun diaqramları ilə  
🔸 **Vaxt Analizi** – ay və il üzrə satış trendi (Line Chart və Column Chart)  
🔸 **Filtrlər və Slicer-lər** – məhsul növü, tarix, region və s. üzrə interaktiv analiz  
🔸 **Drill-down və Drill-through** imkanları

## 🧠 Texnologiyalar

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Məlumat mənbəyi: daxili və/və ya Excel/CSV əsaslı

## 📂 Fayllar

| Fayl adı        | Təsviri |
|-----------------|---------|
| `FINAL.pbix`    | Əsas Power BI dashboard faylı |
| `DataModel`     | Məlumat cədvəlləri və DAX ölçüləri |
| `Report`        | Vizual elementlərin və layout-un saxlandığı fayl |
| `Connections`   | Məlumat mənbəyi bağlantıları |

## 🔧 İstifadə Təlimatı

1. Power BI Desktop proqramını açın.
2. Bu repodakı `FINAL.pbix` faylını daxil edin.
3. İstədiyiniz sahələri və filterləri seçərək analizə başlayın.

## 🎯 Layihənin Məqsədi

Satış performansını real vaxtda və tarixi məlumatlarla vizual təhlil etmək, rəhbərlərin və analitiklərin qərarvermə prosesini asanlaşdırmaq.
